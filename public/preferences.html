LAClient Preferences

When a change is made on this page, its state is automatically saved to your device

Auto Refresher

Modify the automatic refresh functionality of this application

Enabled

When checked, the AskBCS student queue will automatically refresh

Refresh Interval

The amount of time (in seconds) between automatic refreshes

second(s)
Audio Notification

Modify the audio notification functionality for when a new AskBCS question is detected

Enabled

When checked, if a new question is found in the queue, an alert will be played

Audio File

The audio file to be played when a new question is found in the queue

Timeout

The amount of time (in seconds) that should elapse before another alert is played

second(s)
Playback Volume

The volume at which the audio file will be played at (range is from 0.0 to 1.0)
